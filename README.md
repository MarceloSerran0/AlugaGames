<h1> Aluguel de Jogos </h1>

<h2> Sobre </h2>
<p> Projeto desenvolvido para praticar JavaScript com HTML e CSS através do treinamento da Alura. Esse projeto simula um aluguel de jogos dentro de um site
onde a pessoa pode alugar ou devolver o jogo. A lógica utilizada é bem simples porem precisa se atentar a todos os detalhes, principalmente quando utilizamos
funções.</p>
